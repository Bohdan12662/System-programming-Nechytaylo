Що таке GNU/Linux?

Linux — це сімейство Unix-подібних операційних систем, які використовують ядро ​​Linux розробки фінно-американського програміста Лінуса Торвальдса. ОС, що використовують ядро ​​Linux, називаються дистрибутивами Linux, і є такими ж операційними системами як Microsoft Windows або Apple macOS, але з однією дуже важливою особливістю — їх вихідні коди є відкритими, оскільки вони поширюються під ліцензією GNU GPL, яка передбачає створення вільного та відкритого програмного забезпечення (open source software). Це означає, що будь-який користувач має право вивчати та змінювати вихідний код. Офіційним талісманом Linux з 1996 року є пінгвін Tux (скор. від “Torvalds UniX“). Ідею використання пінгвіна як талісману Linux висунув творець цього ядра Лінус Торвальдс. У спільноті програмістів існує суперечка щодо назви операційних систем, які використовують ядро Linux і програмне забезпечення, розроблене під ліцензією GNU GPL. Оскільки ядро Linux саме по собі не є операційною системою, то багато хто вважає коректним використовувати термін «GNU/Linux».

Перелік популярних дистрибутивів Linux:

  • Arch Linux: Дистрибутив, що надає великий функціонал кастомізації. Ідеальний для досвідчених користувачів, які бажають встановити мінімальний набір програм і самостійно налаштувати свою систему.
  • Debian: Один з найстаріших і найстабільніших дистрибутивів. Програмне забезпечення проходить ретельне тестування, що робить його надійним вибором для розробників.
  • Raspbian: Дефолтна ОС для Raspberry Pi, яка підходить для початківців завдяки своїм освітнім ресурсам та простим інструментам програмування.
  • Gentoo: Мета-дистрибутив, який дозволяє користувачам завантажувати та компілювати вихідний код під свої потреби, забезпечуючи високу продуктивність та оптимізацію.
  • Ubuntu: Дуже популярний дистрибутив для початківців, що пропонує зручний інтерфейс і регулярні оновлення, а також підтримує пакування додатків у форматі snaps.
  • Fedora: Дистрибутив, похідний від Red Hat, відомий своєю легкістю установки та доступом до різних інструментів для розробників.
  • OpenSUSE: Спеціально розроблений для розробників, цей дистрибутив включає інструменти для налаштування та автоматизації, такі як YaST.
  • CentOS: Безкоштовна версія Red Hat, що забезпечує стабільність та доступ до інструментів розробки для професіоналів.
  • Solus: Дистрибутив із моделлю контрольованого плаваючого релізу, який дозволяє користувачам отримувати регулярні оновлення без необхідності повного оновлення ОС.
  • Puppy Linux: Дуже легкий дистрибутив, що завантажується в RAM, ідеальний для старих комп'ютерів. Має підтримку різних мов програмування.

Приклади команд роботи в Linux CLI

                
    pwd 
      Показує повний шлях до поточної директорії.
    
    echo [текст] 
      Виводить текст або змінну в термінал.
    
    ls 
      Показує список файлів і каталогів у поточній директорії.
    
    cd [каталог] 
      Змінює поточну директорію на вказану.
    
    touch [файл] 
      Створює новий порожній файл або оновлює час модифікації існуючого.
    
    mkdir [каталог] 
      Створює новий каталог з вказаною назвою.
    
    cp [джерело] [призначення] 
      Копіює файли або каталоги з одного місця в інше.
    
    mv [старе_ім'я] [нове_ім'я] 
      Переміщує або перейменовує файли або каталоги.
    
    rm [файл] 
      Видаляє файл або каталог (додайте -r для видалення каталогів).
    
    history 
      Показує історію введених команд у терміналі.
